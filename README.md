## Installation

### Clone
```bash
$ git clone https://github.com/reihnagm/React-Redux-Hiring-Channel-Website.git
$ cd React-Redux-Hiring-Channel-Website
$ npm install
```
---
### Start Development Server
```bash
$ npm run start
```
